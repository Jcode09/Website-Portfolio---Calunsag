OS - First Come First Serve

Info

The project "First Come First Serve Operating System", a scheduling simulator that will demonstrate the operation of this approach. The goal of this challenge is to modify the First Come First Serve simulator so that it gives is a scheduling algorithm to execute queued tasks, processes, and requests efficiently and automatically by the order of their arrival.
