Laboratory Inventory Project

Info

The project “Laboratory Tools and Equipment Management System” is a system to borrow an inventory of laboratory equipment and tools. It will keep materials including the laboratory and borrower, if item is borrowed. Should be able to accept queries and display results. The system can create, read, update, and display data (CRUD) and uses JOptionPane at the Graphic User Interface (GUI) of the system.
